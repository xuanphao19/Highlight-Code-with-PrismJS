Highlight Code with PrismJS

Bước 1: Nhúng thẻ <link> code CSS của PrismJS:

   <link rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/themes/prism-okaidia.min.css" />

Bước 2: Nhúng code JavaScript của PrismJS:

    <script src="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/components/prism-core.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/plugins/autoloader/prism-autoloader.min.js"></script>

Bước 3: Thêm class="language-html" cho thẻ <code>:

    <code class="language-html">
      <img src="https://htmlcss.fullstack.edu.vn/assets/f8_icon.png"
        width="100"
        height="100"
        alt="F8"/>
    </code>
